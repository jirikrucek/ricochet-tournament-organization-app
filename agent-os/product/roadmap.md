# Product Roadmap

## Phase 1: MVP (Current)

The initial release includes core tournament management capabilities:

- **Tournament Management**: Create and manage multiple tournaments with status tracking (setup → live → finished)
- **Player Registration**: Register and manage players with country flags, ELO ratings, and profiles
- **Match Scoring**: Enter match scores, track micro-points, and assign matches to courts
- **Bracket Visualization**: Interactive Winners and Losers bracket display with zoom capabilities
- **Live View for Spectators**: Real-time tournament updates, match schedules, and standings
- **Multi-language Support**: Interface available in 5 languages (Polish, English, German, Dutch, Czech)
- **Dual-mode Storage**: Works with or without cloud database (Supabase/localStorage)

## Phase 2: Post-Launch

Future enhancements to be determined based on user feedback and tournament organizer needs.

Potential areas for expansion:
- Advanced player statistics and historical performance tracking
- Tournament templates and presets
- Enhanced mobile experience
- Data export and reporting features
- Multi-tournament comparison tools
